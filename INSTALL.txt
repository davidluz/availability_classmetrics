INSTALAÇÃO DO PLUGIN AVAILABILITY CLASS METRICS
=============================================

REQUISITOS:
- Moodle 4.5 ou superior
- PHP 7.4 ou superior

INSTALAÇÃO:

Método 1 - Via Interface Web:
1. Faça login como administrador no Moodle
2. Vá para Administração do Site > Plugins > Instalar plugins
3. Faça upload do arquivo availability_classmetrics.zip
4. Siga as instruções na tela

Método 2 - Via FTP/SSH:
1. Extraia o arquivo availability_classmetrics.zip
2. Copie a pasta 'availability_classmetrics' para:
   [diretório_moodle]/availability/condition/
3. Acesse Administração do Site > Notificações
4. Clique em "Atualizar banco de dados do Moodle"

CONFIGURAÇÃO:
1. Vá para Administração do Site > Plugins > Disponibilidade > Gerenciar condições de disponibilidade
2. Certifique-se de que "Class Metrics" está habilitado

USO:
1. Edite qualquer atividade ou seção
2. Expanda "Restringir acesso"
3. Clique "Adicionar restrição"
4. Selecione "Métricas da Turma"
5. Configure as condições desejadas

DESINSTALAÇÃO:
1. Vá para Administração do Site > Plugins > Visão geral dos plugins
2. Encontre "availability_classmetrics"
3. Clique em "Desinstalar"
4. Confirme a desinstalação

Para mais informações, consulte o arquivo README.md
